# Azeroy Garden
### Feito por Isabella Barbosa

Este jogo se passa na vila de Azeroy, uma vila pacífica e deslumbrante, porem com poucos habitantes. Porém, um dia, monstros sanguinários atacaram o jardim da vila, considerado divino e angelical de tão belo, e se esconderam dentro das masmorras que há debaixo da vila. Seu objetivo será derrotá-los e vingar os habitantes da vila, utilizando uma de três armas disponíveis, uma espada, um arco ou um cajado. Eles estão contando com você, boa sorte!
  
- **Controles:**  
"w" - Move o jogador para cima e altera o símbolo para "^"  
"a" - Move o jogador para a esquerda e altera o símbolo para "<"  
"s" - Move o jogador para baixo e altera o símbolo para "v"  
"d" - Move o jogador para a direita e altera o símbolo para ">"  
"i" - Interage com o objeto que está à frente do jogador  
"o" - O jogador realiza um ataque na célula à sua frente  
  
- **Símbolos e seus significados:**  
"<" - Jogador olhando para a esquerda  
"^" - Jogador olhando para cima  
">" - Jogador olhando para a direita  
"v" - Jogador olhando para baixo  
"*" - Parede: o jogador não pode passar  
"#" - Espinho: o jogador morre ao passar por cima  
"k" - Caixa: o jogador não pode passar, mas pode ser destruída com ataque  
"O" - Botão: executa uma ação ao ser pressionado  
"D" - Porta fechada: o jogador não pode passar  
"@" - Chave: abre uma porta fechada ao interagir  
"=" - Porta aberta: o jogador pode passar  
"L" - Escada: leva o jogador para a próxima fase  
"X" - Monstro Tipo 1  
"Y" - Monstro Tipo 2  
"Z" - Boss Final
